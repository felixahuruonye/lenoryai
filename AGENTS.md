# Project Instructions

## Supabase Setup
To enable the Exam History and Performance Review features, you **must** run the SQL commands found in `supabase_schema.sql` in your Supabase project's SQL Editor. 

This will create the `exams` table with the correct permissions.

## Exam Simulation
The exam questions are generated using the Gemini AI. If you see a "Cloud Error", ensure your `GEMINI_API_KEY` is correctly set in the environment.
